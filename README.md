## Popis sítě
Tento projekt obsahuje model LAN sítě v Cisco Packet Traceru. Síť je rozdělena na dvě části propojené switchi S1 a S2. K S1 jsou připojena klientská PC, k S2 jsou připojeny servery poskytující služby DHCP, DNS a WEB. Konfigurace switchů byla provedena přes Laptop připojený konzolovým kabelem.
